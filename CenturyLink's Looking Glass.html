A Looking Glass is a piece of software running on a web server that allows external users to get a look at routing and network behavior as it originates from the remote network. A looking glass accesses a remote router and performs either a ping, trace, or one of several show commands allowing a view of the IP and BGP route tables.The information is then returned as a web page.
